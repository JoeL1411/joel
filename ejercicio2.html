Galería - Programación Web

HTML

Es el lenguaje que estructura el contenido de una página web.

CSS

Permite dar estilo, colores y diseño a las páginas web.

JavaScript

Hace que las páginas web sean dinámicas e interactivas.

Frontend

Es la parte visual que el usuario ve e interactúa.

Backend

Se encarga de la lógica, bases de datos y servidor.

Responsive

Diseño adaptable a celulares, tablets y computadoras.

Frameworks

Herramientas como React o Angular facilitan el desarrollo.

Git

Sistema de control de versiones usado por programadores.
